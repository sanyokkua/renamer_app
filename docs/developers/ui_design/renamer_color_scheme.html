Section 05

JavaFX CSS Implementation

Ready-to-use JavaFX CSS code. Each block targets a specific zone of the application. Apply to your main Scene via scene.getStylesheets().add(getClass().getResource("theme.css").toExternalForm()).

JavaFX CSS Notes JavaFX uses -fx- prefixed properties. Gradients use linear-gradient() with from ... to ... syntax. Colors can be defined as hex, rgb(), or derived colors using ladder() and derive().
/* ==================================================== RENAMER APP — theme.css Blue-Grey Pastel System · JavaFX CSS ==================================================== */ /* 1. ROOT — global color variables and base */ .root { -fx-base: #EFF4FA; -fx-background: #EFF4FA; -fx-control-inner-background: #FFFFFF; -fx-accent: #2C6FBF; -fx-focus-color: #4A90D9; -fx-faint-focus-color: rgba(74, 144, 217, 0.15); -fx-text-base-color: #162030; -fx-text-inner-color: #162030; -fx-font-family: "System"; -fx-font-size: 13px; } /* 2. LEFT CONFIG PANEL — VBox with style class "config-panel" */ .config-panel { -fx-background-color: #E3EBF5; -fx-border-color: #BDD0E8; -fx-border-width: 0 1 0 0; -fx-padding: 14 16 14 16; -fx-spacing: 8; } /* Section label inside config panel */ .config-panel .section-label { -fx-text-fill: #6688AA; -fx-font-size: 11px; -fx-font-weight: bold; } /* Live preview box inside config panel */ .preview-box { -fx-background-color: #D4E5F9; -fx-border-color: #A8C5E8; -fx-border-width: 1; -fx-border-radius: 5; -fx-background-radius: 5; -fx-padding: 8 10 8 10; } .preview-box .preview-value { -fx-text-fill: #1F3C5C; -fx-font-family: "Monospaced"; -fx-font-weight: bold; } /* 3. TABLE VIEW — header, rows, states */ .table-view { -fx-background-color: #FFFFFF; -fx-border-color: #C4D4E8; -fx-border-width: 0; } .table-view .column-header-background { -fx-background-color: #1F3C5C; } .table-view .column-header { -fx-background-color: #1F3C5C; -fx-border-color: #2C4E72; -fx-border-width: 0 1 0 0; -fx-size: 34px; } .table-view .column-header .label { -fx-text-fill: #EAEEF2; -fx-font-weight: bold; -fx-font-size: 12px; } /* Default row */ .table-row-cell { -fx-background-color: #FFFFFF; -fx-border-color: transparent transparent #D8E5F2 transparent; -fx-border-width: 0 0 0.5 0; -fx-cell-size: 30px; } /* Alternating row */ .table-row-cell:odd { -fx-background-color: #F4F8FD; } /* Hover */ .table-row-cell:hover { -fx-background-color: #E8F0FB; } /* Selected row */ .table-row-cell:selected { -fx-background-color: #D4E5F9; } .table-row-cell:selected .table-cell { -fx-text-fill: #162030; } /* Focused table (not selected) */ .table-view:focused .table-row-cell:selected { -fx-background-color: #D4E5F9; } /* Status row styles (add via pseudoclass or style class in Java) */ .table-row-cell.status-success { -fx-background-color: #E6F4EC; } .table-row-cell.status-error { -fx-background-color: #FEF2F2; } .table-row-cell.status-warning { -fx-background-color: #FEF9EC; } /* Empty rows below data */ .table-row-cell:empty { -fx-background-color: #FFFFFF; } .table-row-cell:empty:odd { -fx-background-color: #F4F8FD; } /* Table cell text */ .table-cell { -fx-text-fill: #35506A; -fx-alignment: CENTER-LEFT; -fx-padding: 0 8 0 8; } /* 4. METADATA PANEL — HBox or GridPane with style class */ .metadata-panel { -fx-background-color: #EAF0F8; -fx-border-color: #C4D4E8; -fx-border-width: 1 0 0 0; -fx-padding: 12 16 12 16; } .metadata-panel .meta-key { -fx-text-fill: #6688AA; -fx-font-size: 11px; -fx-font-weight: bold; } .metadata-panel .meta-value { -fx-text-fill: #162030; -fx-font-family: "Monospaced"; -fx-font-size: 11.5px; } /* 5. BOTTOM ACTION BAR */ .action-bar { -fx-background-color: #F7F9FC; -fx-border-color: #C4D4E8; -fx-border-width: 1 0 0 0; -fx-padding: 8 12 8 12; -fx-alignment: CENTER-RIGHT; -fx-spacing: 8; } /* 6. BUTTONS */ .button-primary { -fx-background-color: #2C6FBF; -fx-text-fill: #FFFFFF; -fx-font-weight: bold; -fx-background-radius: 5; -fx-padding: 6 18 6 18; -fx-cursor: hand; } .button-primary:hover { -fx-background-color: #1E5A9E; } .button-primary:pressed { -fx-background-color: #184D88; } .button-primary:disabled { -fx-background-color: #D8E5F2; -fx-text-fill: #8FA4BB; } .button-secondary { -fx-background-color: #FFFFFF; -fx-text-fill: #2C6FBF; -fx-border-color: #2C6FBF; -fx-border-width: 1.5; -fx-border-radius: 5; -fx-background-radius: 5; -fx-padding: 5 16 5 16; } .button-secondary:hover { -fx-background-color: #EBF2FF; } .button-ghost { -fx-background-color: transparent; -fx-text-fill: #5e7388; -fx-border-color: #C4D4E8; -fx-border-width: 1; -fx-border-radius: 5; -fx-background-radius: 5; } /* 7. COMBOBOX / DROPDOWN */ .combo-box { -fx-background-color: #FFFFFF; -fx-border-color: #BDD0E8; -fx-border-width: 1; -fx-border-radius: 4; -fx-background-radius: 4; } .combo-box:focused { -fx-border-color: #4A90D9; -fx-effect: dropshadow(gaussian, rgba(74,144,217,0.25), 4, 0, 0, 0); } .combo-box-popup .list-view { -fx-background-color: #FFFFFF; -fx-border-color: #BDD0E8; } .combo-box-popup .list-cell:filled:hover { -fx-background-color: #E8F0FB; } .combo-box-popup .list-cell:filled:selected { -fx-background-color: #D4E5F9; -fx-text-fill: #1F3C5C; } /* 8. RADIO BUTTON & CHECKBOX */ .radio-button .radio { -fx-border-color: #BDD0E8; -fx-background-color: #FFFFFF; } .radio-button:selected .radio { -fx-border-color: #2C6FBF; -fx-background-color: #2C6FBF; } .radio-button .text, .check-box .text { -fx-text-fill: #35506A; } .check-box .box { -fx-border-color: #BDD0E8; -fx-background-color: #FFFFFF; -fx-background-radius: 3; } .check-box:selected .box { -fx-border-color: #2C6FBF; -fx-background-color: #2C6FBF; } /* 10. SCROLL BAR */ .scroll-bar { -fx-background-color: #EFF4FA; } .scroll-bar .thumb { -fx-background-color: #BDD0E8; -fx-background-radius: 4; } .scroll-bar .thumb:hover { -fx-background-color: #94B0CC; } /* 11. SEPARATOR */ .separator .line { -fx-border-color: #C8D8EE; -fx-border-width: 0.5; } /* 12. SPLIT PANE DIVIDER */ .split-pane > .split-pane-divider { -fx-background-color: #BDD0E8; -fx-padding: 0 1 0 1; } .split-pane > .split-pane-divider:hover { -fx-background-color: #4A90D9; } /* 13. STATUS BADGE LABELS — add via getStyleClass() */ .badge-pending { -fx-background-color: #EBF1FA; -fx-text-fill: #5580AC; -fx-border-color: #BDD0E8; -fx-border-width: 0.5; -fx-background-radius: 10; -fx-border-radius: 10; -fx-padding: 2 8 2 8; } .badge-success { -fx-background-color: #E6F4EC; -fx-text-fill: #2E6E50; -fx-border-color: #A8D5BC; -fx-border-width: 0.5; -fx-background-radius: 10; -fx-border-radius: 10; -fx-padding: 2 8 2 8; } .badge-error { -fx-background-color: #FEF2F2; -fx-text-fill: #B42222; -fx-border-color: #F5AAAA; -fx-border-width: 0.5; -fx-background-radius: 10; -fx-border-radius: 10; -fx-padding: 2 8 2 8; } .badge-warning { -fx-background-color: #FEF9EC; -fx-text-fill: #9A6B15; -fx-border-color: #F5D98A; -fx-border-width: 0.5; -fx-background-radius: 10; -fx-border-radius: 10; -fx-padding: 2 8 2 8; }

Section 07

Accessibility Contrast Reference

All text-on-background combinations are checked against WCAG 2.1 contrast requirements. These must pass AA (4.5:1 for normal text, 3:1 for large/bold text).

Text Color Background Approx. Ratio WCAG Grade Usage
#162030 #FFFFFF ~14.5:1 AAA ✓ Table cell primary text
#35506A #EFF4FA ~7.1:1 AAA ✓ Labels on app background
#EAEEF2 #1F3C5C ~10.8:1 AAA ✓ Table column headers
#FFFFFF #2C6FBF ~5.0:1 AA ✓ Button label on primary btn
#6688AA #EAF0F8 ~3.8:1 AA (lg) ✓ Meta keys — use 11px bold
#2E6E50 #E6F4EC ~5.6:1 AA ✓ Success badge text
💡

The muted label color #6688AA on light blue backgrounds sits just at AA for large text. Always use it at minimum 11px bold (600 weight) for metadata keys. For body text, use #35506A or darker. Use the free Colour Contrast Analyser tool to spot-check after implementation.
